About

This is a website made to predict the classes of the imagenet dataset. The model used in background is Resnet-18 where resnet stands for residual network. Specifically, In this website we used torchvision pretrained resnet-18 model which is earlier trained on imagenet dataset. We used some sort of transfer learning in order to do the desired task. Resnet-18 is a model made on architecture which is decided by many experiments. It contains many layers like Convolutional layer, Flatten Layer, Average Pool layer, Dense Layer and many more. We can furthur modify this model using our dataset and after changing some Layers.

In this website you can predict the output of the output classes of imagenet by two ways:
1. By directly uploading the image by upload button present there on top left side of the page. In this you have to simply select the image from your local drive and then submit the predict button it will give the output which have highest probabilty of matching which the outputted class.
2. Second one is by clicking the image directly from your camera and then predict the image. The above button click and predict after clicking this button it will automatically click the image and then predict the output related with this image.
